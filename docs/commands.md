
# Полезные Команды

## Собрать докер файл и запустить

Эти команды нужны для проверки верного деплоя,
так как он происходит через Dockerfile

```
docker build . -t redata
```

```
docker run -p 8080:8080 redata
```

## CLI для дерева выражений

```
dotnet run --project src/ReData.ExprTree.Cli
```

Пример ввода: `Sum(a + b)`.
После ввода выражения CLI сразу рендерит `SVG` через Graphviz из NuGet-пакета и открывает файл.
Для многострочного выражения завершите ввод строкой `;;`.
В `SVG` отображается:
- один узел с исходным текстом выражения;
- дерево выражения;
- связь от узла исходного текста к корню дерева.
- нижний слой токенов финального PostgreSQL SQL;
- пунктирные связи от SQL токенов к узлам дерева, из которых они были скомпилированы.
