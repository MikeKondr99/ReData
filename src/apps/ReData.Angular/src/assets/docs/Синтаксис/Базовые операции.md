
# Базовые операции

## Арифметические операторы

Сложение
`a + b`

Вычитание
`a - b`

Умножение
`a * b`

Деление
`a / b`

Отрицание числа
`-5`
`-(a + b)`


## Операторы сравнения

Равенство
`a = b`

Неравенство
`a != b`

Больше/меньше
`a > b`
`a < b`

Больше или равно/меньше или равно
`a >= b`
`a <= b`


## Логические операторы

Логическое И
`condition1 and condition2`

Логическое ИЛИ
`condition1 or condition2`

Отрицание не обладает специальным синтаксисом и нужно использовать функцию 'Not'
`Not(condition)`
