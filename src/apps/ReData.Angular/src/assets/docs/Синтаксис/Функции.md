
# Функции

## Обычный синтаксис

`ИмяФункции(аргумент1, аргумент2, ...)`

Примеры:
- `Floor(column1)`
- `Date('2025-05-08')`
- `Concat(FirstName, ' ', LastName)`
- `Pi()`
- `AddMonths(MyDate, 2)`

## Синтаксис методов объектов

`аргумент1.ИмяФункции(аргумент2, ...)`

Примеры:
- `column1.Floor()`
- `'2025-05-08'.Date()`
- `MyDate.AddMonths(2)`
- `[Description].Substring(2, 5)`

## Взаимозаменяемость синтаксисов

Оба синтаксиса эквивалентны:

Эти выражения идентичны
`Text(id)`
`id.Text()`

// Эти выражения идентичны
Upper(FirstName)
FirstName.Upper()

## Цепочки вызовов

"Методный" синтаксис удобен для цепочек:

Традиционный синтаксис (менее читаемо)
`IsNull(EmptyIsNull(Substring([Description], 2, 5)))`

Методный синтаксис (более читаемо)
`[Description].Substring(2, 5).EmptyIsNull().IsNull()`
