

# Имена полей

Имена полей используются для обращения к данным хранящимся в столбце в таблице данных

## Простые имена

Если имя поля содержит только 
- буквы латинского или русского языка
- цифры 
- нижнее подчеркивание `_`

То можно просто написать название этого поля
- `column1`
- `total_sum`
- `price`
- `quantity`

## Имена со специальными символами

Если имя поля не подходит под 'простое' то можно использовать альтернативный синтаксис

- `[2023 Sales]`
- `[Имя сотрудника]`
- `[Date Created]`
- `[Unit Price ($)]`
- `[Я люблю \] в полях]` ']' пишется внутри имени как '\\]'
