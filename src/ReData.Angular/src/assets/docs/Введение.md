# Введение

Язык выражений позволяет создавать вычисления для фильтрации и преобразования данных с помощью интуитивного синтаксиса.

## Основные возможности

- **Вычисления**: математические операции с числами и датами
- **Фильтрация**: условия для отбора данных
- **Преобразования**: изменение формата и структуры данных
- **Агрегация**: группировка и статистические вычисления
- **Интерполяция**: динамическое создание текста

## Быстрый старт

Простое вычисление
`(Price * Quantity) * 0.8`

Условие для фильтрации данных
`Status = 'active' and Balance > 1000`

Преобразование строк
`FirstName + ' ' + Upper(LastName)`
