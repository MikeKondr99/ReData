
# Литералы

Литералы - это постоянные значения в выражениях.

## Целые числа:
- `1`
- `-45`
- `1000`

## Дробные числа (всегда с точкой):
- `123.45`
- `-0.5`
- `0.001`

## Строковые литералы

- `'Привет'`
- `'Hello world'`
- `'2024-01-15'`

## Булевы литералы

Представляют собой константное значение истины или лжи

- `true` Истина
- `false` Ложь

## Создание дат

У дат нет литералов и для их создания используются функции

Неправильно
`2025-05-08`

Правильно
- `Date('2025-05-08')`
- `'2025-05-08'.Date()`
- `Date(2025)`
- `Date(2025, 5)`
- `Date(2025, 5, 8)`
